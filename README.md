🏏 IPL 2025 Dashboard

 Overview
An interactive Power BI dashboard that analyzes the IPL 2025 season using visual insights. The dashboard helps users explore team performance, player statistics, match results, and tournament trends.

Features
- Team-wise performance analysis
- Player statistics and rankings
- Match results and points table
- Interactive filters and slicers
- Clean and user-friendly dashboard

Tools Used
- Power BI
- DAX
- Power Query
- Excel/CSV Dataset

 Project Files
- `IPL_2025_DASHBOARD.pbix` – Power BI dashboard file

How to Use
1. Download the `.pbix` file.
2. Open it using Microsoft Power BI Desktop.
3. Explore the dashboard using the available filters and visuals.
